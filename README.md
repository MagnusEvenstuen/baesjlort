# Baesjlort
## Cloning the project
The contents of this project can be cloned using either SSH, and HTTPS.

HTTPS:
```bash
  git clone --recurse-submodules https://github.com/MagnusEvenstuen/baesjlort.git
```

SSH:
```bash
  git clone --recurse-submodules git@github.com:MagnusEvenstuen/baesjlort.git
```

## Running the files
The files needs to be built before it can be run. To build the files go to the folder location, and open a terminal. Then run it using:
```bash
colcon build
```
Then the repository needs to be sourced to be able to access the packages. It is only needed to do ones per open terminal. This is done using:
```bash
source install/setup.bash
```
